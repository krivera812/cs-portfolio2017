Important Vocab

    HTML- Hyper Text Markup Language (uses <>)
  • <title>-the main name of a creators page (
  • <body>body-where the creator's content goes
  • <head>-not seen on the page but contains important
    information for the browser.
  • <h>-ranging from 1-6, it is the tag resposible for headers
  • <p>-makes paragraph texts
  • links-connect your page to another source
  • <img>-pictures that go into the page
  • ol/ul (ordered list/unordered list)-list items with bullets or numbers
    • li (list item)-the items in the list
    CSS-Cascading Style Sheet (uses {} )
  • class/id-define certain tags so they can be identified differently to similar tags
  • color- changes the color of a specific tag
  • aligin-determines the placement of a box

Navbar

One thing that I learned how to do recently was how to install a navbar. The navbar helps a viewer find their way around the site. To start the process, I created a CSS file under the name navbar.css. Back in HTML I managed to link it in the head under title in order for them to work together.

<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="ie=edge">
<title>About Me</title>
<link rel="stylesheet" href="navbar.css" type="text/css"/>
</head>

I made an unordered list, with the id "navbar" (under the <h1>). Underneath that, I added the links to the sites that connected to About Me. It ended up looking like this:

    

<h1> About Me</h1>
<ul id="navbar">
<li> <a href="index.html"> About Me</a> </li>
<li><a href="aboutmehowto.hmtl"> How to-About Me </a></li>
</ul>

    

In CSS, I used a id attribute that corresponded with the id name in html. (#navbar). I then algined the text to the center for the site to look balanced. I then created the borders around the the navbar buttons inorder to seperate them. This included adding the border color and px, the margin, and the border radius.

#navbar { text-align: center
;
}
#navbar li {
border: 2px solid black;
margin: 5px;
display:inline-block;
background-color: white;
padding:5px;
border-radius:15px;
}

I also ended up adding a hover color just for fun, with the rgb color 255, 237,240.

#navbar li:hover {background-color:rgb(255, 237, 240);}
